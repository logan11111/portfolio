Projects

Scotts Outdoor Creations - Online website


In this project, I created a website that allows easy tracking of online orders for a small home buisness. I used a combination of html, css, and javascript to accomplish my goals. One challenge I ran into included using the swiper.js library to make the pages more dynamic which took some research into how the library operated. Submitting Forms and sending emails to orders placed was another big hurdle to overcome as there are many ways to do so but there also limiting factors to consider. Formspree was my final choice for this matter as it worked seamlessly with the current setup. Like most software, this project is still evolving according to the clients needs.
